Acceptrics is a free, customizable consent banner for your website, helping you manage user consent for Google cookies.

== Description ==

Acceptrics Consent Banner is a free, customizable consent banner for your website. This plugin allows you to define your own JavaScript to manage user consent for Google cookies. Acceptrics is compatible with wp-consent-api to ensure consent is signaled correctly.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/acceptrics-consent-banner` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings -> Acceptrics Consent Banner screen to configure the plugin.

== Frequently Asked Questions ==

= How do I customize the consent banner? =

You can customize the consent banner by navigating to the plugin settings page and entering your custom JavaScript.

= Does this plugin support Google EU UCP compliance? =

Yes, this plugin is designed to help you comply with Google's EUUCP by providing a customizable consent banner.

== External services ==

This plugin connects to the Acceptrics server, it's needed to show the Acceptrics banner on your webpage.

It loads the banner code from the Acceptrics server. No sensitive information or PII is shared with Acceptrics.
